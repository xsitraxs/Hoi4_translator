# 🌍 HoI4 Localization Tool

Автоматизированный инструмент для перевода и оптимизации файлов локализации (.yml) игр на движке Paradox (Hearts of Iron IV, Stellaris и др.). Программа сочетает в себе гибкость Python и производительность Java-сервисов для корректной обработки игровых тегов.

---

## 🚀 Основные возможности

* **Интеллектуальный перевод:** Поддержка нескольких движков (Google Translate, DeepL, MyMemory) через `deep-translator`.
* **Защита синтаксиса Paradox:** Автоматическое экранирование и сохранение игровых плейсхолдеров:
    * Переменные: `$VALUE$`, `[GetTag]`
    * Цветовые коды: `§R`, `§W`, `§!`
    * Иконки: `£money`, `£trigger_icon`
    * Спецсимволы: `\\n`, `\\t`
* **Гибридная архитектура:** Использование Java через `JPype` для сложной обработки строк и асинхронные запросы `aiohttp` для ускорения сетевых операций.
* **Умное кэширование:** База данных SQLite хранит все выполненные переводы, экономя трафик и время при повторном запуске.
* **Продвинутый GUI:** Интерфейс на `tkinter` с логами в реальном времени, прогресс-барами и предпросмотром перед сохранением.

## 🛠 Технические требования

Для корректной работы проекта необходимы:

1.  **Python 3.9+**
2.  **Java JDK 11+** (для работы Java-сервиса через JPype)
3.  **Зависимости:**
    ```bash
    pip install -r requirements.txt
    ```

## ⚙️ Особенности форматирования (Важно)

Инструмент настроен на строгое соблюдение структуры файлов локализации:
* **Без лишних заголовков:** По умолчанию исключается добавление `l_russian` или других языковых меток в начало файла, если это не требуется.
* **Индентация:** Строгое соблюдение одного пробела в начале строк перевода.
* **Кодировка:** Принудительное использование `UTF-8 BOM`, необходимого для корректного чтения файлов движком игры.

## 📦 Установка и запуск

1.  Клонируйте репозиторий.
2.  Убедитесь, что `HoI4Translator.java` скомпилирован или доступен для загрузки через JPype.
3.  Запустите основной скрипт:
    ```bash
    python hoi4_translator.py
    ```
