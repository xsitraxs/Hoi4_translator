# HOI4 Localisation Translator

Программа с графическим интерфейсом для автоматического перевода локализации модов Hearts of Iron IV с английского на русский язык через Google Translate.

![Python](https://img.shields.io/badge/Python-3.8+-blue) ![License](https://img.shields.io/badge/License-MIT-green)

---

## Что делает

- Копирует все `.yml` файлы из папки `english` оригинального мода
- Переименовывает файлы: `_l_english.yml` → `_l_russian.yml`
- Заменяет заголовок `l_english:` → `l_russian:` внутри файлов
- Переводит строки пачками (батчами) через Google Translate — быстро
- Сохраняет плейсхолдеры нетронутыми (`$VAR$`, `[Scope.Var]`, `§Y`, `£icon` и др.)
- Пропускает уже переведённые строки с кириллицей (опционально)
- Игнорирует `languages.yml` чтобы не сломать список языков в игре
- Показывает прогресс и лог в реальном времени

## Установка

### Вариант 1 — EXE (рекомендуется, Python не нужен)

Скачай `HOI4_Localisation_Translator.exe` со страницы [Releases](../../releases/latest) и просто запусти.

### Вариант 2 — Python

**1. Клонируй репозиторий:**
```bash
git clone https://github.com/xsitraxs/hoi4-localisation-translator.git
cd hoi4-localisation-translator
```

**2. Установи зависимости:**
```bash
pip install -r requirements.txt
```

**3. Запусти:**
```bash
python hoi4_translator.py
```

## Использование

### Шаг 1 — Укажи папки

| Поле | Что указывать |
|------|---------------|
| **Папка english** | Папка `localisation/english` оригинального мода |
| **Папка russian** | Папка `localisation/russian` твоего мода-перевода |

Папка russian создастся автоматически если её нет.

### Шаг 2 — Настрой размер батча

**Размер батча** — это количество строк, которые отправляются в Google Translate за один запрос вместо того чтобы переводить каждую строку отдельно. Это ускоряет перевод в 20–30 раз.

| Значение | Когда использовать |
|----------|--------------------|
| **10–20** | Google выдаёт ошибки или капчу |
| **40** | Стандарт, подходит в большинстве случаев |
| **60–80** | Если всё стабильно и хочешь ещё быстрее |

> Чем больше батч — тем быстрее перевод, но выше риск что Google заблокирует запрос. Если видишь ошибки в логе — уменьши батч до 20.

### Шаг 3 — Галочка "Пропускать уже переведённые строки"

Если в папке `russian` уже есть файлы с частичным переводом — поставь галочку. Программа найдёт строки с кириллицей и оставит их нетронутыми, переведя только оставшиеся на английском.

### Шаг 4 — Нажми "Начать перевод"

В логе будет показано:
- Какой файл обрабатывается сейчас
- Сколько строк переведено в каждом файле
- Сколько строк пропущено (уже переведены)
- Итоговая статистика по всем файлам

## Важно

- Перевод автоматический — качество может быть неидеальным. Рекомендуется пройтись по файлам и поправить ошибки вручную.
- Программа не трогает `languages.yml` — этот файл отвечает за список языков в игре, его изменение может сломать мод.
- Плейсхолдеры (`$VAR$`, `[Scope.Var]`, `§Y`, `£icon` и др.) сохраняются в оригинальном виде и не переводятся.
