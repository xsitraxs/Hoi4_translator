# 🗺️ HOI4 Localisation Translator

Профессиональный инструмент с графическим интерфейсом для автоматического, безопасного и быстрого перевода локализации модов Hearts of Iron IV. 

![Python](https://img.shields.io/badge/Python-3.8+-blue) ![License](https://img.shields.io/badge/License-MIT-green) ![Version](https://img.shields.io/badge/Version-1.3.0-brightgreen)

---

## 🔥 Что нового в версии 1.3.0
- **Идеальное форматирование HOI4:** Программа автоматически удаляет старые языковые заголовки (`l_english:`, `l_russian:`) и пишет ключи строго с одним пробелом в начале (формат ` KEY:0 "Text"`). Ваш мод больше не будет ломать движок Paradox!
- **Неубиваемая база данных (SQLite):** Кэш переводов теперь хранится в скрытой папке пользователя (`~/.hoi4_translator_data/`). Вы можете удалять или обновлять `.exe` файл — ваша база наработанных переводов никуда не исчезнет.
- **Умный авто-фикс и валидация:** Программа следит, чтобы машинный перевод не "съел" важные плейсхолдеры (например, `$VAL$`, `§Y`). Если переводчик их потерял, скрипт попытается автоматически их восстановить.
- **Постобработка русской типографики:** Автоматически убирает лишние пробелы перед знаками препинания, исправляет кавычки на ёлочки («») и корректирует заглавные буквы после цветовых кодов `§`.
- **Поддержка Java (JPype):** Добавлена возможность интеграции кастомных Java-библиотек (`hoi_logic.jar`) для сложной обработки.

---

## ⚡ Возможности

- Поддержка движков: **Google Translate** (сверхбыстрый Async API), **DeepL**, **MyMemory**, **JavaSim** (кастомный).
- **Проверка API ключа DeepL** — встроенный валидатор ключей с выводом точной причины ошибки (лимит исчерпан, неверный ключ и т.д.).
- Перевод с любого языка (английский, немецкий, французский) на русский.
- Полностью игнорирует `languages.yml`.
- Многопоточный асинхронный перевод (`aiohttp`) — обрабатывает тысячи строк за секунды.
- Защита от бана по IP (умная система Retry с экспоненциальной задержкой).

## 🛠️ Установка

### Вариант 1 — EXE (Рекомендуется, Python не нужен)
Скачай `HOI4_Translator_PRO.exe` со страницы [Releases](../../releases/latest) и запусти. Программа работает как Portable-версия, все данные сохраняет в `C:\Users\ИмяПользователя\.hoi4_translator_data\`.
