# 🗺️ HOI4 Localisation Translator

Программа с графическим интерфейсом для автоматического перевода локализации модов Hearts of Iron IV.

![Python](https://img.shields.io/badge/Python-3.8+-blue) ![License](https://img.shields.io/badge/License-MIT-green) ![Version](https://img.shields.io/badge/Version-2.0-brightgreen)

---

## Возможности

- Поддержка трёх движков перевода: **Google Translate**, **DeepL**, **MyMemory**
- **Проверка API ключа DeepL** — встроенная кнопка для валидации ключа перед запуском
- Перевод с любого языка на русский (английский, немецкий, французский, испанский, польский)
- **Асинхронный перевод** — использует `aiohttp` для ускорения работы с Google Translate
- **Кэширование в SQLite** — все переведённые строки сохраняются в базе данных, повторные запросы не требуются
- Батч-перевод — отправляет несколько строк за один запрос, в 20–30 раз быстрее поштучного
- Сохраняет плейсхолдеры нетронутыми (`$VAR$`, `[Scope.Var]`, `§Y`, `£icon`, `\n` и др.)
- Пропускает уже переведённые строки (определяет по наличию кириллицы + проверка кэша)
- Игнорирует `languages.yml` — не ломает список языков в игре
- Работает с модами на любом исходном языке, не только английском
- Сохраняет все настройки между запусками
- Показывает прогресс, скорость перевода и подробный лог
- Кнопка "Стоп" — останавливает перевод в любой момент
- **Заготовка под Java-интеграцию** — возможность подключения внешнего Java-сервиса перевода

## Установка

### Вариант 1 — EXE (рекомендуется, Python не нужен)

Скачай `HOI4_Pro_Translator.exe` со страницы [Releases](../../releases/latest) и запусти.

### Вариант 2 — Python

**1. Клонируй репозиторий:**
```bash
git clone https://github.com/xsitraxs/hoi4-localisation-translator.git
cd hoi4-localisation-translator
```

**2. Установи зависимости:**
```bash
pip install -r requirements.txt
```

**3. Запусти:**
```bash
python hoi4_translator.py
```

## Использование

### Шаг 1 — Укажи папки

| Поле | Что указывать |
|------|---------------|
| **Оригинал (english)** | Папка `localisation/english` оригинального мода |
| **Перевод (russian)** | Папка `localisation/russian` твоего мода-перевода |

Папка russian создастся автоматически если её нет.

### Шаг 2 — Выбери движок

| Движок | Особенности |
|--------|-------------|
| **Google** | Бесплатно, быстро, асинхронный режим. Рекомендуется для большинства случаев |
| **DeepL** | Лучшее качество перевода, требует API ключ (есть бесплатный тариф). **Есть кнопка проверки ключа!** |
| **MyMemory** | Бесплатно, переводит поштучно, есть лимиты |

### Шаг 3 — Проверка API (для DeepL)

Если выбран **DeepL**:
1. Вставь API ключ в поле
2. Нажми **"Проверить API"**
3. Дождись результата:
   - ✓ **Валиден** — ключ рабочий, можно начинать перевод
   - ✗ **Неверный ключ** — проверь правильность ввода
   - ✗ **Лимит исчерпан** — закончились бесплатные запросы
   - ✗ **Нет сети** — проверь подключение к интернету

**Кнопка "НАЧАТЬ ПЕРЕВОД" будет работать только после успешной проверки!**

### Шаг 4 — Настрой размер батча

**Батч** — количество строк за один запрос к переводчику. Влияет только на Google Translate.

| Значение | Когда использовать |
|----------|--------------------|
| **1–10** | Google выдаёт ошибки или капчу |
| **20** | Стандарт, стабильно работает |
| **40–80** | Если всё стабильно и нужна максимальная скорость |

### Шаг 5 — Галочки

- **Пропускать переведенное** — строки с кириллицей не будут переводиться заново. Удобно если прерывал перевод или хочешь дополнить уже частично переведённый мод
- **Открыть папку по итогу** — после завершения откроет папку russian в проводнике

### Шаг 6 — Нажми НАЧАТЬ

В логе будет показано:
- Какой файл обрабатывается
- Сколько строк переведено / пропущено
- Текущая скорость (строк/сек)
- Итоговая статистика

## Структура мода-перевода

```
my_translation_mod/
├── descriptor.mod
└── localisation/
    └── russian/
        ├── _metro_events_l_russian.yml
        ├── _metro_units_l_russian.yml
        └── ...
```

**descriptor.mod** должен содержать зависимость от оригинального мода:
```
dependencies = { "Название оригинального мода" }
```

## Важно

- Перевод автоматический — качество может быть неидеальным, особенно у Google. Рекомендуется проверить результат вручную
- Плейсхолдеры (`$VAR$`, `[Scope.Var]`, `§Y` и др.) сохраняются в оригинальном виде
- Программа не трогает `languages.yml`
- При использовании DeepL бесплатный API ключ можно получить на [deepl.com](https://www.deepl.com/pro-api)
- **Кэш переводов** хранится в файле `translation_cache.db` в папке программы
- **Логи** сохраняются в файл `hoi4_translator.log`

## Технические детали

- **Асинхронность**: при установленной библиотеке `aiohttp` перевод через Google работает в асинхронном режиме, что значительно ускоряет обработку больших объёмов текста
- **Кэширование**: используется SQLite база данных для хранения всех переведённых строк. Повторный перевод той же строки с тем же движком происходит мгновенно
- **Java-интеграция**: при установке `jpype1` доступна заготовка для подключения внешнего Java-сервиса перевода (режим `JavaSim`)
